VerityRefiner Job Results: Import

job result file:  IRS_Migration_1_analysis_20240131T094344.dat

Introduction

This Notebook imports data from a VerityRefiner job results export file. This file has a special structure to allow importing it into a variety of analysis tools, including MicroSoft Excel and Jupyter Notebook. However, the import must be done knowing the detailed file structure and what each section means and should be presented. Technik Interlytics provides information on the file structure, how to export it, and how to import it in its help guide.

This Notebook uses the export from a sample job and creates Python objects for each of the major results components. It then uses these objects to filter into DataFrames to generate many tables and figures showing the result data. These roughly correspond to how results are shown in VerityRefiner.

To use this Notebook, change the first cell to point to the export file to read in. Then, either run the entire Notebook or use the debugger to step through code cells to see what is happening or if you get errors to track down their root causes.

The completed Notebook can be exported using nbconvert. Refer to its documentation for how that is done but an example (not guaranteed) terminal command is:

jupyter nbconvert --to html --TemplateExporter.exclude_input=True .\VerityRefiner_Results.ipynb
